# CPU 模拟器

根据计算机组成与原理的课程设计要求编写的 cpu 模拟器，可以读取特定的汇编指令集文件，并以执行一条微指令为最小单位进行单步执行和全部执行。

使用了 vue cli3 + vue2 + electron6

> 示例汇编代码：[example/test.data](example/test.data)
